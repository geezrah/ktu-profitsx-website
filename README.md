# KTU HIGH PROFITSX

A stunning, fully responsive trading platform landing page with live animated market data, 70/30 profit share breakdown, and social media integration.

## Features

✨ **Live Animation Engine** — Real-time canvas-based 3D candlestick charts with market ticker  
📱 **Responsive Design** — Auto-detects desktop/mobile and adapts layout seamlessly  
🎨 **Professional UI** — Bold typography, glow effects, and market-grade aesthetics  
📊 **Interactive Elements** — Clickable social links (Instagram, Telegram, WhatsApp)  
🚀 **Performance Optimized** — Fast load times, smooth animations, no dependencies

## Getting Started

### Local Development

1. **Clone or download** the repository
2. **Open `index.html`** in your web browser
3. The site will auto-detect desktop/mobile mode and adjust accordingly

No build tools, no dependencies — just pure HTML, CSS, and vanilla JavaScript.

### Deploy to Netlify

1. **Push to GitHub** (see instructions below)
2. Go to [netlify.com](https://netlify.com) and connect your GitHub repo
3. Netlify will auto-deploy on every push

Or manually deploy:
- Drag & drop the folder into Netlify's deploy area
- Your site will be live in seconds

## Custom Domain

1. In Netlify dashboard, go to **Domain settings**
2. Click **Add custom domain**
3. Point your domain's DNS to Netlify
4. Netlify will auto-provision an SSL certificate
